# Based on drawr-web-app
added some material design goodness

## to make it run:
1. clone
2. npm install
3. npm install https://github.com/drawr-team/drawrlib
4. node app.js
5. fun!

## current bugs:
- Window needs to be resized once for the canvas to have correct size. Has height 0 before.
